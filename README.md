# Leetcode submissions

## How to navigate this repo
Here's how this repo is laid out.

1. Solutions are inside of the folder titled `my-folder`.
2. Opening it, will lead you to see a list of problem names.
3. When you open a specific problem it will contain 2 items
   * `README.MD` -> Containing the problem description
   * `solution.java` -> This is my written Java solution
